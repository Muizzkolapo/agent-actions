## 0.1.12 - 2026-04-19
### Enhancement or New Feature
* Release v0.1.11
* Add self-reflection feedback strategy for reprompt retries (use_self_reflection config option)
* Add LLM critique escalation for stubborn reprompt failures — after N failed attempts, a separate LLM call analyzes why validation keeps failing and includes that analysis in the next retry prompt
### Under the Hood
* Unified online and batch reprompt logic via shared helpers (parse_reprompt_config, safe_validate)
### Bug Fix
* Drop directives now correctly exclude fields from passthrough wildcards by extracting passthrough before drop and applying drop to both observe and output contexts.
* Warn when inline schema shadows schema_name; add missing json_mode warning to AgacClient
* Pass compiled schemas to Groq, Cohere, and Mistral APIs for structured output enforcement instead of silently ignoring them
* Retry transient OpenAI 400 JSON parsing errors instead of failing immediately; guard response_format when schema is None
* Cascade failures now report as blocked instead of independent errors; batch-mode actions show wall-clock duration from submission to completion
* Cross-workflow records now deliver content and propagate source_guid through downstream pipelines
* Reprompt batch resolves source.* observe fields via storage backend; --downstream defers workflows when parent is paused for batch
* FILE-mode output pipeline: fix lineage collision when inputs share source_guid by removing false default-to-0 in _reattach_source_guid
* Batch resubmission loop: prevent submit_batch_job from resubmitting completed batches when upstream guard filters records with on_false: skip
* Downstream tool actions no longer crash on absent fields from guard-filtered upstream records; schema validation auto-fixes guard-nullable fields to accept null
* Prevent non-serializable data (NaN, Infinity, datetime, bytes, sets) from producing invalid JSON in OpenAI request payloads
* dispatch_task() in prompt templates now resolves UDF functions instead of passing the literal string to the LLM
